موكب أمير المؤمنين (ع) — تأسّس سنة ١٣٨٤هـ / ١٩٦٤م

هذا الدليل يحدّد قواعد استخدام الشعار، الألوان، والخطوط في جميع المطبوعات والمنصّات الرقمية، ليحافظ الموكب على حضورٍ بصريّ موحّد ومهيب.

٠١

الشعار

الشعار عمل خطّي متكامل: قبّةٌ خضراء تحمل عبارة «السلام عليك يا أبا عبد الله»، يحيط بها اسم الموكب بالخط الديواني الأحمر، وتاريخ التأسيس ١٣٨٤هـ / ١٩٦٤م، على قرصٍ أصفر. يُستخدم الشعار دائمًا كوحدة واحدة لا تتجزّأ.

على الأصفر — الاستخدام الأساسي
على الفاتح — للمطبوعات والوثائق
على الداكن — للرايات والشاشات

المساحة الآمنة

¼ القطر

اترك حول الشعار مساحة فارغة لا تقلّ عن رُبع قطره من كل جهة. لا يجوز أن يزاحمه نصّ أو عنصر آخر داخل هذه المساحة.

الحدّ الأدنى للحجم

٤٨ بكسل — رقمي
٢٠ ملم — طباعة

دون هذه الأحجام تفقد تفاصيل الخط وضوحها، فيُستعاض عنها باسم الموكب كتابةً.

محظورات الاستخدام

  • لا يُعاد تلوين الشعار أو تُبدَّل ألوانه الثلاثة.
  • لا يُمَطّ أو يُضغَط أو يُدار أو تُضاف إليه ظلال وتأثيرات.
  • لا تُفصَل القبّة أو أيّ عبارة خطّية عن بقية الشعار.
  • لا يوضع على خلفيات مزدحمة أو صور تُضعف وضوحه.
  • لا يُحاط بإطار أو زخرفة إضافية غير القرص الأصفر.
٠٢

رموز CSS للتطوير

انسخ هذه المتغيّرات إلى مشروع الموقع (جاهزة للاستخدام مع Claude Code). الخطّان متاحان عبر Google Fonts.

:root {
  /* الألوان */
  --mawkab-yellow: #FEF33E;   /* أصفر الراية — خلفيات وهوية */
  --mawkab-red:    #D70C00;   /* أحمر الخط — عناوين وتأكيد */
  --mawkab-green:  #157201;   /* أخضر القبة — روابط وأيقونات */
  --mawkab-paper:  #FAF7EC;   /* خلفية الصفحات */
  --mawkab-ink:    #201C10;   /* النصوص */

  /* تدرجات مشتقة */
  --mawkab-red-dark:    #A80900;  /* حالة hover للأحمر */
  --mawkab-green-dark:  #0E4F01;  /* حالة hover للأخضر */
  --mawkab-yellow-soft: #FFFBD6;  /* خلفية صفراء خفيفة */

  /* الخطوط */
  --font-heading: 'Amiri', serif;
  --font-body: 'IBM Plex Sans Arabic', system-ui, sans-serif;
}

html { direction: rtl; }
body {
  font-family: var(--font-body);
  color: var(--mawkab-ink);
  background: var(--mawkab-paper);
}
h1, h2, h3 { font-family: var(--font-heading); }
a { color: var(--mawkab-green); }
a:hover { color: var(--mawkab-red); }
